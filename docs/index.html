바로 답을 적으세요.

메신저에 설명을 덧붙이거나, 버그를 공유하거나, 자료를 비교할 때. K-Capture는 필요한 부분을 띄워두고 표시한 다음 바로 복사하거나 저장하게 합니다.

주석복사
필요한 장면만 바로 캡처단축키를 누르고 화면의 필요한 부분만 끌어 선택
참조를 옆에 둔 채 작업캡처를 띄워두고 다른 앱을 계속 보며 비교
상대가 볼 곳만 표시긴 설명 대신 선, 도형, 짧은 메모로 핵심 전달

A focused workflow

보여주고 싶은 화면을,
설명까지 한 번에.

채팅, 문서, 버그 제보를 오가며 캡처를 다시 찾을 필요가 없습니다.

01

필요한 장면만 잡습니다

단축키나 메뉴바에서 시작해, 지금 설명해야 할 부분만 빠르게 선택합니다.

02

옆에 띄워두고 봅니다

캡처를 항상 위에 두고 원래 작업 화면과 나란히 비교하거나 작은 글자까지 살펴봅니다.

03

상대가 볼 곳을 짚습니다

펜, 도형, 화살표, 텍스트로 핵심을 짚고, 보여주면 안 되는 정보는 블러나 모자이크로 가립니다.

04

바로 붙여 넣거나 보냅니다

완성한 화면을 대화창에 복사하고, 파일이 필요할 때만 PNG로 저장해 전달합니다.

More than a screenshot

캡처 한 장으로,
설명이 끝나게.

말로 길게 적기 어려운 화면 상황을, 필요한 도구로 바로 정리합니다.

화면 속 문장을 다시 쓰지 마세요

캡처에 보이는 문장을 선택해 필요한 부분만 텍스트로 가져옵니다.

복잡한 모양도 필요한 부분만

네모로 자르기 애매한 영역도 손으로 따라 선택해 남기거나 덜어냅니다.

상품이나 인물만 따로 보내세요

피사체를 찾아 배경을 빼고, 필요한 대상만 투명 PNG로 만듭니다.

가릴 곳은 남기지 마세요

이름, 알림, 개인 정보처럼 공유하면 안 되는 부분을 블러나 모자이크로 가립니다.

나중에 다시 볼 캡처를 모아두세요

자주 참고하는 화면은 즐겨찾기에 남겨두고 필요할 때 다시 열어 이어서 편집합니다.

새 버전을 놓치지 마세요

업데이트가 준비되면 앱 안에서 알려주므로, 새 기능을 찾으러 따로 돌아다닐 필요가 없습니다.

Annotate with intent

설명은 짧게,
화면 위에서 바로.

긴 문장 대신, 보이는 곳에 선 하나와 짧은 메모를 남겨 상대가 바로 이해하게 합니다.

  • 한눈에 짚어주기펜, 형광펜, 도형, 화살표, 텍스트를 골라 상대가 볼 위치를 분명하게 표시합니다.
  • 살펴보기 편하게작은 글자나 비교할 부분은 확대해 보고, 캡처 창은 필요한 곳으로 옮겨둡니다.
  • 보낼 방법 고르기대화창에는 바로 붙여 넣고, 파일이 필요한 경우에는 PNG로 저장하거나 끌어 전달합니다.

주석 도구

펜 형광펜 도형 화살표 텍스트 블러 모자이크

Cmd+C로 대화창에 바로 붙여 넣고, Cmd+S로 PNG를 남깁니다. 자주 쓰는 도구와 색상은 키보드로 빠르게 바꿀 수 있습니다.

Get started

설치하고,
다음 캡처부터 바로.

  1. 최신 DMG를 다운로드합니다릴리스 페이지에서 최신 .dmg 파일을 받습니다.
  2. Applications 폴더로 옮깁니다DMG를 열고 K-Capture를 Applications 폴더로 드래그합니다.
  3. 처음 한 번 열기를 허용합니다처음 실행이 차단되면 앱을 우클릭한 뒤 ‘열기’를 선택합니다.
  4. 화면 기록 권한을 켭니다macOS 시스템 설정에서 K-Capture의 화면 기록을 허용하면 캡처를 시작할 수 있습니다.

처음 한 번만 확인하세요

미서명 앱 경고가 보이면 ‘열기’를 한 번 선택하면 됩니다. “손상되어 열 수 없습니다”라는 메시지가 나오는 경우에만 아래 명령을 사용하세요.

xattr -cr /Applications/K-Capture.app
