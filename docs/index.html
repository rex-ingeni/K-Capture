K-Capture

화면 일부를 캡처해 항상 위에 떠 있는 플로팅 창으로 띄우고, 그 위에 도형·텍스트·블러 주석을 다는 macOS 캡처 앱.

⬇︎ 다운로드 (macOS) 모든 버전
Apple Silicon · Intel · macOS 12+ · 무료

무엇을 할 수 있나

✂️

영역 캡처

단축키로 화면 어디든 드래그해 원하는 영역만 잘라냅니다.

📌

플로팅 창

캡처를 항상 위에 떠 있는 창으로 띄워 다른 작업과 나란히 봅니다.

✏️

주석

펜·도형·화살표·텍스트로 캡처 위에 바로 표시를 남깁니다.

🌫️

블러 · 가리기

민감한 부분을 블러/모자이크로 가려 안전하게 공유합니다.

📋

복사 · 저장

클립보드로 바로 복사하거나 이미지 파일로 저장합니다.

🔄

자동 업데이트

새 버전이 나오면 앱 안에서 바로 내려받아 교체합니다.

설치 방법

  1. 다운로드위 버튼으로 최신 .dmg 를 받습니다.
  2. 응용 프로그램으로 이동DMG 를 열고 K-Capture 를 Applications 폴더로 드래그합니다.
  3. 처음 실행앱을 우클릭 → 열기 로 실행합니다. (미서명 앱이라 최초 1회만 필요)
  4. 메뉴바에서 사용상단 메뉴바 아이콘 또는 단축키로 캡처를 시작합니다.

이후 업데이트는 앱 우상단 버전 버튼(또는 메뉴바 · 업데이트 알림)에서 한 번에 처리됩니다 — 다시 내려받을 필요 없습니다.
