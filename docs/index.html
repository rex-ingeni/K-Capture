캡처를
바로 쓰는 화면으로.

K-Capture는 화면 일부를 빠르게 캡처하고, 항상 위에 띄운 채 주석·OCR·피사체 추출·복사와 저장까지 이어 가는 작업용 도구입니다.

macOS 다운로드 Windows 다운로드 GitHub 보기

macOS 12 이상 · Windows 10 20H1 이상 · 무료

필요한 화면은 계속 눈앞에 캡처를 플로팅 창으로 띄우고 바로 표시를 남깁니다.
주석복사
전역 단축키로 시작작업 흐름을 끊지 않고 영역 선택
항상 위 플로팅참조 화면을 다른 앱 위에 유지
캡처 후 바로 편집주석과 추출을 한 캔버스에서 처리

A focused workflow

캡처하고, 정리하고, 전달합니다.

짧은 참고부터 공유용 이미지까지, 필요한 동작만 이어지는 흐름입니다.

01

영역을 고릅니다

전역 단축키 또는 메뉴바에서 시작한 뒤, 필요한 화면만 드래그해 캡처합니다.

02

눈앞에 고정합니다

캡처는 항상 위에 떠 있는 창으로 열립니다. 여러 장을 띄우고 비교하거나, 확대와 팬으로 세부를 살펴볼 수 있습니다.

03

의도를 표시합니다

펜, 형광펜, 도형, 화살표, 텍스트와 블러·모자이크로 설명에 필요한 부분만 남깁니다.

04

바로 사용합니다

클립보드로 복사하거나 PNG로 저장하고, 필요하면 Finder나 다른 앱으로 파일을 바로 끌어냅니다.

More than a screenshot

필요한 후처리를 캡처 위에서.

다른 앱을 열기 전에, 화면에서 바로 해결할 수 있는 기능을 담았습니다.

텍스트 선택 OCR

화면 속 문장을 인식하고 원하는 범위를 드래그해 텍스트로 복사합니다.

올가미 선택

정형화되지 않은 영역도 자유롭게 지정해 남기거나 제거할 수 있습니다.

피사체 추출

자동으로 피사체를 찾아 배경을 제거하고, 필요한 부분만 투명 PNG로 만듭니다.

블러와 모자이크

공유하기 전 민감한 정보나 시선을 분산시키는 부분을 가립니다.

즐겨찾기와 편집

필요한 캡처를 즐겨찾기에 모아 다시 열고, 캔버스와 객체를 이어서 편집합니다.

앱 내 업데이트

새 릴리스가 있으면 메뉴바와 메인 창에서 확인하고 앱 안에서 업데이트합니다.

Annotate with intent

표시만 남기고, 해상도는 지킵니다.

설명에 필요한 도구를 빠르게 바꾸고, 결과는 원본 밀도 기준으로 복사하거나 저장합니다.

  • 주석 도구펜·형광펜·사각형·원·화살표·텍스트와 색상·굵기를 바로 조절합니다.
  • 보기 제어줌과 팬, 좌우·상하 반전, 창 투명도로 참조하기 좋은 상태를 만듭니다.
  • 공유 방식 선택복사, PNG 저장, Shift+드래그 파일 전달 중 현재 작업에 맞는 방식을 고릅니다.

주석 도구

펜 형광펜 도형 화살표 텍스트 블러 모자이크

Cmd+C는 현재 주석을 합성해 복사하고, Cmd+S는 PNG로 저장합니다. 자주 쓰는 도구와 색상은 키보드로도 바로 바꿀 수 있습니다.

Get started

설치는 한 번, 캡처는 언제든.

  1. 최신 DMG를 다운로드합니다릴리스 페이지에서 최신 .dmg 파일을 받습니다.
  2. Applications 폴더로 옮깁니다DMG를 열고 K-Capture를 Applications 폴더로 드래그합니다.
  3. 처음 한 번 열기를 허용합니다처음 실행이 차단되면 앱을 우클릭한 뒤 ‘열기’를 선택합니다.
  4. 화면 기록 권한을 켭니다macOS 시스템 설정에서 K-Capture의 화면 기록을 허용하면 캡처를 시작할 수 있습니다.

처음 실행 안내

미서명 앱 경고가 보이면 ‘열기’를 한 번 선택하세요. “손상되어 열 수 없습니다”라는 메시지가 나오는 경우에만 아래 명령을 사용합니다.

xattr -cr /Applications/K-Capture.app
